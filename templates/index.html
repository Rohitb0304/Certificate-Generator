Instructions:

  • In your PPTX template, include the tag <<FULL NAME>> where you want the student's name to appear.
  • Your CSV file should have a column named Full_Name with the corresponding names of the students.
  • Ensure that the PPTX file is saved in the .pptx format.
  • The limit for certificate generation is 80 records at a time.
  • Each certificate generation will take approximately 30-40 seconds depending on the PPT File Size and data.
  • Upload both files in the correct format to generate the certificates.
